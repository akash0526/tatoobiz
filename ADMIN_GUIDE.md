## Features

### Dashboard (`/admin/dashboard`)
- Overview stats
- Quick action buttons

### Gallery Manager (`/admin/dashboard/gallery`)
- Upload new tattoo photos
- Toggle "Featured" status (shows on homepage)
- Delete photos

### Artists Manager (`/admin/dashboard/artists`)
- Add / Edit / Delete artists
- Toggle active/inactive
- Manage specialties

### Services Manager (`/admin/dashboard/services`)
- Edit pricing, descriptions, duration
- Show/hide services

### Bookings (`/admin/dashboard/bookings`)
- View all booking requests
- Change status (New → Contacted → Booked → Completed)
- Quick call & WhatsApp buttons

### Testimonials (`/admin/dashboard/testimonials`)
- Approve or hide client reviews
